# 🧹 Smart File Organizer (Python)

A simple desktop tool built with Python to automatically organize messy folders.

## 🚀 Features

* 📂 Sorts files into categories (Images, Videos, Docs, etc.)
* ⚡ One-click folder selection
* 🧠 Smart file detection
* 🔁 Handles duplicate file names

## 🛠️ Tech Used

* Python
* Tkinter (GUI)
* OS & Shutil modules

## ▶️ How to Use

1. Download the `.exe` file (for Windows)
2. Run the app
3. Select a folder
4. Click organize 🎉
